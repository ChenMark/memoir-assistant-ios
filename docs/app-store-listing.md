## 七、审核自查清单

### 7.1 App Store 审核指南 5 大问题

- [ ] **1. 安全 — 用户生成内容**: 回忆录/评论/照片有举报机制吗？ → 后端需添加 `report` 接口
- [ ] **2. 性能 — 完整功能**: 是否有死链接或占位功能？ → 全功能完整
- [ ] **3. 业务 — 支付**: 目前无内购，标注"免费"
- [ ] **4. 设计 — 最低要求**: iOS 17+, 支持最新设备分辨率
- [ ] **5. 法律 — 隐私**: 隐私政策 URL 已部署

### 7.2 常见拒审原因预防

| 风险 | 状态 | 预防措施 |
|------|------|----------|
| 缺少隐私政策 | ✅ 已准备 | docs/privacy-policy.md |
| 第三方登录无网页版 | ⚠️ OAuth 登录需有网页版备选 | 确保邮箱登录正常 |
| 缺少账号删除功能 | ✅ 已实现 | AccountSettingsView 中 |
| 崩溃/卡死 | ⚠️ 待 TestFlight 验证 | CrashReportService 已就绪 |
| 权限描述不清晰 | ⚠️ 需检查 Info.plist | 见下方 |
| UI 与系统差距大 | ✅ 拟物化但合规 | 使用了标准 Navigation |

### 7.3 Info.plist 权限描述检查

```xml
<key>NSCameraUsageDescription</key>
<string>忆往昔需要使用相机拍摄照片，记录你的珍贵回忆</string>

<key>NSPhotoLibraryUsageDescription</key>
<string>忆往昔需要访问相册，用于选择和上传老照片</string>

<key>NSMicrophoneUsageDescription</key>
<string>忆往昔需要使用麦克风进行语音输入，帮助你轻松记录回忆</string>

<key>NSSpeechRecognitionUsageDescription</key>
<string>忆往昔需要语音识别功能，让你可以用说话的方式记录回忆</string>

<key>NSFaceIDUsageDescription</key>
<string>使用面容ID快速解锁，保护你的回忆隐私</string>
```
