# 忆往昔 · 隐私政策

> 最后更新：2026年6月17日

## 我们收集的信息

忆往昔（以下简称"我们"）尊重并保护所有用户的个人隐私权。

### 您主动提供的信息
- **账户信息**：邮箱地址、手机号码（用于登录验证）
- **用户内容**：您撰写的回忆录文字、上传的照片
- **语音数据**：语音输入功能采集的音频（仅用于实时转写，不存储原始音频）
- **亲友信息**：您添加的亲友姓名、关系、出生年月等

### 自动收集的信息
- **设备信息**：设备型号、操作系统版本
- **日志数据**：崩溃日志、性能数据（匿名化处理）

## 我们如何使用信息

| 用途 | 说明 |
|------|------|
| 提供核心服务 | 回忆录存储、AI 访谈、照片管理、家族树 |
| 改进产品体验 | 分析匿名化的使用数据以优化功能 |
| 账户安全 | 身份验证、防止滥用 |

## 数据存储与安全

- 您的回忆录内容和照片存储在阿里云 OSS（中国大陆）
- 所有数据传输使用 HTTPS 加密
- JWT Token 存储在设备 Keychain 安全区域
- 我们不存储您的明文密码
- 我们不向第三方出售您的个人信息

## 您的权利

- **访问与导出**：您可以在设置中查看个人资料，回忆录支持分享导出
- **删除账户**：在"设置 → 账户设置 → 注销账号"中可以永久删除所有数据
- **撤销同意**：您可以在系统设置中关闭语音输入、照片访问等权限

## 儿童隐私

本产品主要面向 50 岁以上中老年用户，不面向 13 岁以下儿童。

## 政策更新

我们可能会更新本隐私政策，更新后的政策将在本页面发布。
